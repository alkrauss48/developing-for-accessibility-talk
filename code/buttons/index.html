Buttons & Links

Go Back

An Anchor Tag

<a href="#">Go Back</a>

Best for:

  • Taking you to a different URL / inner page.

A Button Tag

<button type="button">Generate an Alert</button>

Best for:

  • Doing some simple client-side logic.

A Form Submission

<button type="submit">Submit</button>

Best for:

  • Handling user-entered data.
  • Note: This is the default button type.

Tips:

  • If you are only navigating somewhere, always use an anchor tag. If you are doing client-side logic, then use a button.
  • The type property of a button tag is very important; if you use a "button" type for a submission, or a "submit" type for a normal action, those are no-no's.
  • Lastly, make sure your links and buttons have good visual focus indication. Usually your browser will handle this, but often times I like to apply the same "hover" styles for "focus" styles too.
